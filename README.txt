misomiso.exeの使い方

みそみそ〜、とマストドンでつぶやく専用コマンドラインアプリケーションです。

# 使い方

1.

プログラムをダウンロードしてZIPを展開します。

2.

展開したフォルダの中に以下のような内容を config.yaml という名前で保存します。

mastodon の項目は自分のログイン情報に書き換えます。

-- ここから --

data_file: data.yaml
mastodon:
   domain: mstdn.jp
   email: example@example.com
   password: password

-- ここまで --

3.

misomiso.cmd をダブルクリックすると「みそみそ〜」と投稿されます。


4.

何かキーを押せば終了です。
